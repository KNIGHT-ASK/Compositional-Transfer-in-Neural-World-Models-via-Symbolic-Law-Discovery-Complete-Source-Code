# Project Expertise: Compositional World Models

This repository implements cutting-edge research in **Physics-Informed Machine Learning (PIML)**, specifically focusing on the zero-shot composition of physical laws.

## Core Technical Skills

### 1. Tangent Space Residual Superposition
- **Skill**: Theoretical and practical implementation of additive ODE composition.
- **Description**: By training neural networks to predict residuals in the tangent space (Lie Algebra) of physical states, this project demonstrates that isolated laws (e.g., gravity, spring forces) can be superposed linearly without additional training on the combined environment.
- **Implementation**: See `src/composition_experiment.py` and `src/exp_u_2d_orbit.py`.

### 2. SINDy Symbolic Extraction
- **Skill**: Automated discovery of symbolic equations from black-box neural networks.
- **Description**: Utilizing **Sparse Identification of Non-linear Dynamics (SINDy)** and LASSO regression to "distill" trained neural weights into human-readable LaTeX-ready equations.
- **Implementation**: See `src/sindy_extraction.py`.

### 3. Modular Physics-Informed Neural Networks (PINNs)
- **Skill**: Designing architectural boundaries that preserve physical invariants.
- **Description**: Developing decoupled neural "brains" that encapsulate specific symmetries (e.g., height-invariance for gravity). This modularity prevents "catastrophic interference" and enables zero-shot transfer.
- **Implementation**: See `src/setup_part_a.py` through `setup_part_d_json.py`.

### 4. Rigorous Scientific Benchmarking
- **Skill**: Statistical validation of neural ODE trajectories.
- **Description**: Implementing multi-seed rollout verification, Welch’s T-Tests for model superiority, and 100K-transition baseline comparisons to prove research-grade results.
- **Implementation**: See `src/exp_p_full_statistics.py` and `src/exp_i_welchs_superiority.py`.

## Demonstrated Workflows

### Zero-Shot Composition Pipeline
1. **Isolated Training**: Train modular MLP/Neural ODE on a single physical law (Environment A: Gravity).
2. **Tangent Superposition**: Sum the predicted velocity residuals (`dv`) and position residuals (`dy`) from multiple modules.
3. **Trajectory Rolling**: Autoregressively roll out the composed model in a target environment (Environment C: Combined) without seeing any training data.

### Symbolic Distillation Workflow
1. **Neural Probing**: Sample a trained module across a dense state-space grid.
2. **Sparse Regression**: Fit a polynomial feature library to the probe data using LASSO.
3. **Constant Recovery**: Extract physical constants (e.g., $g \approx 9.8$, $k \approx 2.0$) directly from the sparse coefficients.
